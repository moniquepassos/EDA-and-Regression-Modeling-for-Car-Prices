# Car Price Prediction using Machine Learning

Projeto de Machine Learning focado na previsão de preços de carros utilizando técnicas de regressão com Python e Scikit-Learn.

O projeto inclui análise exploratória de dados (EDA), preprocessing, feature engineering e comparação entre diferentes modelos de regressão.

## Objetivos

- Realizar análise exploratória dos dados (EDA)
- Tratar valores ausentes e outliers
- Aplicar feature engineering
- Construir pipeline de preprocessing
- Comparar modelos de regressão
- Avaliar performance usando R² e RMSE

## Pipeline do Projeto

1. Coleta e carregamento dos dados
2. Análise exploratória (EDA)
3. Tratamento de valores ausentes
4. Tratamento de outliers
5. Feature engineering
6. Encoding de variáveis categóricas
7. Normalização e preprocessing
8. Treinamento dos modelos
9. Avaliação dos resultados

## Modelos Testados

- Linear Regression
- Random Forest Regressor
- SVR
- Decision Tree

## Aprendizados

Durante este projeto, pratiquei:

- análise exploratória de dados
- preprocessing
- feature engineering
- tratamento de outliers
- pipelines com Scikit-Learn
- avaliação de modelos de regressão
- interpretação de métricas como R² e RMSE

## Resultados

O melhor modelo alcançou aproximadamente:

R²: 0.73
RMSE: 182312.7

Os resultados demonstram boa capacidade do modelo em identificar padrões nos dados e realizar previsões próximas aos valores reais.
